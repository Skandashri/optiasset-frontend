ROLE: Unauthenticated User (Authentication Only)
--------------------------------------------------------------------------------

PAGE: /login (Authentication Entry Point)

  COMPONENT: <AuthLayout />
    - Centers login form with enterprise branding
    - Responsive container with logo and title
    - Theme toggle for light/dark mode preference

  COMPONENT: <LoginForm />
    - Email input with validation (required, email format)
    - Password input with visibility toggle
    - Submit button with loading state
    - Error display for invalid credentials

  COMPONENT: <SSOButtonGroup /> (Future Enhancement)
    - Google OAuth, Microsoft Azure AD, GitHub integration

  COMPONENT: <ForgotPasswordLink /> (Future Enhancement)
    - Password recovery navigation


ROLE: IT Administrator (Full System Access)
--------------------------------------------------------------------------------

PAGE: /dashboard (Main Admin Landing Page)

  COMPONENT: <Sidebar />
    - Primary navigation menu with RBAC filtering
    - Shows all links: Dashboard, Inventory, Users, Assignments, Reports
    - Collapsible for mobile, logout button at bottom

  COMPONENT: <TopNavbar />
    - Logged-in user profile, notifications, theme toggle, logout dropdown

  COMPONENT: <StatCards />
    - 4 metric cards: Total Assets, Active Assignments, Available Inventory,
      Total Employees (with trend indicators)

  COMPONENT: <RecentAssignmentsTable />
    - Last 5-10 asset assignments/returns with sortable columns

  COMPONENT: <PendingIssuesWidget /> (Future)
    - Open support tickets requiring admin action


PAGE: /inventory (Asset Management)

  COMPONENT: <Sidebar /> - Reused from dashboard

  COMPONENT: <TopNavbar /> - Reused from dashboard

  COMPONENT: <InventoryFilterHeader />
    - Search bar (asset tag, model name)
    - Status filter (All, Available, Assigned, Maintenance, Retired)
    - Category filter (Laptops, Monitors, Peripherals, Furniture)

  COMPONENT: <AddAssetButton />
    - Opens AddAssetModal, requires manage:assets permission

  COMPONENT: <AssetDataTable />
    - Paginated table with columns: Asset Tag, Model/Name, Category, Status,
      Current Holder, Actions
    - Row selection, bulk actions, inline edit (future)

  COMPONENT: <ExportReportDropdown />
    - Export options: CSV, Excel, PDF with column selection

  MODALS:
    - <AddAssetModal />: Form to create new assets
    - <EditAssetModal />: Update asset details
    - <ViewAssetModal />: Read-only detailed view
    - <DeactivateConfirmModal />: Soft delete confirmation


PAGE: /users (User Management)

  COMPONENT: <Sidebar /> - Reused

  COMPONENT: <TopNavbar /> - Reused

  COMPONENT: <UserFilterHeader />
    - Search by name/email, role filter, status filter

  COMPONENT: <AddUserButton />
    - Opens AddUserModal, requires manage:users permission

  COMPONENT: <UserDirectoryTable />
    - Columns: Name, Email, Role, Status, Join Date, Actions
    - Row actions: View, Edit, Change Role, Deactivate

  MODALS:
    - <AddUserModal />: Create new user account
    - <EditUserModal />: Update user information
    - <UserRoleManagementModal />: Change role assignment

PAGE: /report-issue (Submit Support Ticket)

  COMPONENT: <EmployeeSidebar /> - Reused

  COMPONENT: <TopNavbar /> - Reused

  COMPONENT: <IssueSubmissionForm />
    - Multi-step form:
      Step 1: Select Asset (from my assignments)
      Step 2: Issue Category (Hardware, Software, Damage, Other)
      Step 3: Description (textarea with min characters)
      Step 4: Upload Images (drag-drop, max 5)
      Step 5: Review & Submit
    - Progress indicator, save as draft option

  COMPONENT: <CancelButton />
    - Returns to previous page with unsaved changes warning


PAGE: /issues (Track Submitted Issues)

  COMPONENT: <EmployeeSidebar /> - Reused

  COMPONENT: <TopNavbar /> - Reused

  COMPONENT: <IssueStatusTrackerTable />
    - Columns: Ticket ID, Asset, Issue Type, Submitted Date, Status
    - Status color coding: Open (Blue), In Progress (Yellow),
      Resolved (Green), Closed (Gray)
    - Sortable columns

  COMPONENT: <IssueDetailModal />
    - Full ticket timeline, admin responses, resolution notes

  - Clear selection, custom renderer

COMPONENT: <ModalOverlay />

COMPONENT: <DataTablePagination />
